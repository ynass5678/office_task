# Информационная система управления личными задачами сотрудника офиса

## Описание
Платформа для управления личными задачами сотрудников офиса.  
Содержит инструменты создания, редактирования и контроля задач, управления пользователями, отслеживания статусов выполнения и администрирования системы.

---

## Установка

1. Установить Python 3.10+ и MySQL

2. Установить зависимости:  
   pip install -r requirements.txt

3. Создать базу данных:  
   CREATE DATABASE office_tasks;

4. Настроить подключение к базе данных в файле db.py

5. Запустить приложение:  
   python app.py

---

## Тестовые пользователи

- Админ: admin / admin123  
- Сотрудник 1: user / user123  
- Сотрудник 2: test2 / test123  

---

## Основные возможности

- Создание и назначение задач сотрудникам  
- Просмотр списка задач с фильтрацией и поиском  
- Изменение статуса задачи (новая, в работе, выполнена и др.)  
- Установка приоритета и категории задач  
- Редактирование задач  
- Управление пользователями (админ-панель)  
- Изменение ролей пользователей  
- Отслеживание сроков выполнения задач  

---

## Обзор

Платформа предоставляет пользователям удобный интерфейс для работы с задачами.

### Сотрудники могут:
- Просматривать назначенные им задачи  
- Создавать новые задачи  
- Редактировать и изменять их статус  

### Администратор имеет расширенные возможности:
- Управление пользователями  
- Назначение ролей (администратор / сотрудник)  
- Просмотр статистики по задачам  
- Контроль активности пользователей  

Система построена по клиент-серверной архитектуре и обеспечивает хранение данных  
в реляционной базе данных с использованием нормализованной структуры.
